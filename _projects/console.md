## Launcher code

```bash
# In $HOME/steam.sh

xrandr --setprovideroffloadsink 0xd7 0x43
DRI_PRIME=1 SDL_VIDEO_MINIMIZE_ON_FOCUS_LOSS=0 /usr/games/steam %U
```
